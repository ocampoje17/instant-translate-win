# Instant Translate for Windows

Instant Translate là app Windows giúp bạn dịch nhanh đoạn văn bản đang chọn sang ngôn ngữ đích cố định chỉ với một phím tắt.

App phù hợp khi bạn cần viết bình luận/chat bằng ngôn ngữ khác từ ngôn ngữ chính bạn dùng hằng ngày (ví dụ: viết tiếng Việt và dịch nhanh sang tiếng Anh để đăng Reddit, X/Twitter, nhắn tin với người nước ngoài...).

## Điểm nổi bật

- Dịch nhanh bằng hotkey toàn cục (mặc định: `Ctrl + Shift + E`)
- Native app cho Windows, chạy nhẹ, có thể chạy nền
- Chọn provider linh hoạt: `Gemini` hoặc `OpenAI-compatible`
- Cài đặt gọn, dễ dùng cho người phổ thông
- API key và dữ liệu lịch sử lưu cục bộ trên máy
